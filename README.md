## Deskripsi Singkat Project
Project ini bertujuan untuk mengimplementasikan operasi dasar pada sinyal 1D dan citra 2D menggunakan Python. Operasi yang dilakukan meliputi penjumlahan, penggeseran, amplifikasi, serta pengujian sistem linier berdasarkan sifat homogenitas dan additivitas.

## Library yang Digunakan
- NumPy
- Matplotlib
- OpenCV (cv2)

## Cara Menjalankan Notebook
1. Buka file `operasi_sinyal_citra.ipynb` menggunakan Google Colab atau Jupyter Notebook.
2. Jalankan setiap cell secara berurutan mulai dari Bagian A hingga Bagian D.
3. Amati hasil visualisasi dan analisis yang diperoleh.

## Ringkasan Hasil Eksperimen
Operasi penjumlahan, penggeseran, dan amplifikasi berhasil diterapkan pada sinyal 1D dan citra 2D. Pengujian sistem menunjukkan bahwa T(x)=2x memenuhi sifat homogenitas dan additivitas sehingga merupakan sistem linier, sedangkan T(x)=x² merupakan sistem non-linier.

## Kesimpulan Singkat
Operasi dasar pada sinyal dan citra merupakan fondasi penting dalam Pengolahan Sinyal Digital. Selain itu, konsep sistem linier sangat penting karena menjadi dasar berbagai aplikasi pengolahan sinyal dan citra yang lebih kompleks.
